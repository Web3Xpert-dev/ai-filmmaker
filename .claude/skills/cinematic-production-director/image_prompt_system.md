# IMAGE PROMPT SYSTEM

Every image prompt must include:

- complete character identity
- age
- facial structure
- hairstyle
- wardrobe details
- emotional expression
- body posture
- environment details
- cinematic lighting
- atmosphere
- camera composition
- lens style
- cinematic realism
- environmental texture
- depth layering
- cinematic color atmosphere

Prompts must feel production-grade.

Prompts must be visually precise.

Prompts must avoid generic wording.

Prompts must avoid vague cinematic language.

Always prioritize:
- realism
- cinematic immersion
- emotional clarity
- environmental detail
- visual coherence
