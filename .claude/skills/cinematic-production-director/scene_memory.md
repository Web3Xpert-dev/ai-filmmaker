# SCENE MEMORY

Persist memory between all scenes.

Track:
- character appearance
- wardrobe state
- emotional arc
- weather state
- environment damage
- object positions
- physical injuries
- camera language style
- lighting continuity
- cinematic tone
- sound atmosphere

When generating future scenes:
- inherit all established details
- evolve naturally from previous context
- avoid resets
- avoid redesigning characters

The same person must always remain visually recognizable.
