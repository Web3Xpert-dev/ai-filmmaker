# ORCHESTRATION RULES

Transform every user request into:

Idea
→ narrative expansion
→ emotional structure
→ scene segmentation
→ cinematic shot planning
→ continuity inheritance
→ image prompt generation
→ motion prompt generation
→ cinematic sound direction
→ final production package

Every scene must have:
- purpose
- tension
- emotional movement
- cinematic intention
- narrative progression

Never create disconnected scenes.

Scene transitions must feel motivated.

Cinematic pacing must evolve gradually.

Escalate emotional intensity naturally.
